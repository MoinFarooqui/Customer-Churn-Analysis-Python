# 📊 Exploratory Data Analysis (EDA) Summary

## Project Objective

The objective of this project is to explore the Telco Customer Churn dataset and identify patterns related to customer demographics, service subscriptions, contract types, payment methods, and customer churn using descriptive statistics and data visualization techniques.

---

# Dataset Overview

* Dataset: Telco Customer Churn
* Records: 7,043 customers
* Features: 21 columns
* Target Variable: Churn

---

# EDA Workflow

## 1. Data Loading

* Imported the dataset using Pandas.
* Loaded the dataset into a DataFrame.
* Displayed sample records using `head()` and `tail()`.

---

## 2. Data Inspection

* Checked dataset dimensions.
* Examined column names and data types.
* Generated summary statistics.
* Verified missing and duplicate values.

---

## 3. Data Preparation

* Converted `TotalCharges` to numeric format.
* Replaced blank values where necessary.
* Converted `SeniorCitizen` values into categorical labels (Yes/No).

---

## 4. Customer Churn Analysis

* Analyzed customer churn distribution.
* Created count plots.
* Generated a pie chart showing the percentage of churned and retained customers.

---

## 5. Demographic Analysis

Analyzed customer distribution based on:

* Gender
* Senior Citizen Status
* Partner
* Dependents

---

## 6. Service Usage Analysis

Explored customer subscriptions for:

* Phone Service
* Multiple Lines
* Internet Service
* Online Security
* Online Backup
* Device Protection
* Tech Support
* Streaming TV
* Streaming Movies

---

## 7. Contract & Payment Analysis

Analyzed:

* Contract Types
* Paperless Billing
* Payment Methods

---

## 8. Customer Tenure Analysis

* Examined tenure distribution.
* Compared tenure between churned and retained customers.

---

## 9. Visualization Techniques

The following visualizations were used:

* Count Plots
* Histograms
* Pie Charts
* Subplots
* 100% Stacked Bar Charts

---

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

# Conclusion

The exploratory data analysis provided a comprehensive understanding of customer demographics, service subscriptions, and behavioral patterns associated with customer churn. These findings establish a strong foundation for future predictive modeling and business intelligence reporting.
